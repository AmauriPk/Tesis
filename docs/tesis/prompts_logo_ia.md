# Prompts para Generación del Logo de SIRAN con IA

## Contexto del sistema
SIRAN es un prototipo de sistema integrado de reconocimiento de aeronaves no tripuladas (drones) mediante visión artificial, desarrollado como tesis de ingeniería. La interfaz actual del sistema usa tema verde neón sobre fondo negro.

---

## Prompt 1 — Español (uso general)

```
Logo minimalista y profesional para un sistema tecnológico de detección de drones llamado SIRAN. El diseño debe incluir un ojo estilizado con una retícula de detección interna, dentro de la cual aparece la silueta simplificada de un dron cuadricóptero (cuatro aspas). Estilo geométrico limpio, sin gradientes complicados. Paleta de colores: verde neón (#22C55E) sobre fondo negro profundo (#0A0A0A). Texto "SIRAN" debajo en tipografía sans-serif bold. No incluir armas, banderas, escudos oficiales ni símbolos militares reales.
```

---

## Prompt 2 — Inglés (uso general)

```
Minimalist and professional logo for a drone detection system called SIRAN. The design features a stylized eye with a detection reticle inside, containing a simplified quadcopter drone silhouette. Clean geometric style, flat design, no complex gradients. Color palette: neon green (#22C55E) on deep black background (#0A0A0A). Text "SIRAN" below in bold sans-serif typography. No weapons, flags, official shields, or real military symbols.
```

---

## Prompt 3 — Versión vectorial (SVG/Illustrator)

```
Vector logo design for SIRAN drone detection system. Geometric eye shape with crosshair/reticle overlay, minimalist quadcopter drone silhouette inside target frame. Flat vector illustration, no raster effects, no gradients, SVG-compatible. Two-color scheme: neon green #22C55E and white #FFFFFF on transparent background. Bold sans-serif logotype "SIRAN" aligned below the symbol. Clean anchor points, no unnecessary nodes.
```

---

## Prompt 4 — Icono pequeño (favicon 32x32 / 64x64)

```
App icon 64x64 pixels for a drone surveillance system named SIRAN. Extremely simple: a circle outline containing a small drone silhouette (4-rotor X shape) framed by thin corner bracket markers []. Flat design, neon green on black, no text, legible at 32x32 pixels. Single layer, no shadows, no gradients.
```

---

## Prompt 5 — Fondo transparente (PNG)

```
Logo for SIRAN system on transparent background. Stylized surveillance eye with drone detection reticle. Minimalist geometric design. Color: neon green #22C55E. No background fill. High resolution 1024x1024 PNG. Suitable for overlaying on dark or light backgrounds. No text included in this version.
```

---

## Prompt 6 — Versión monocromática (blanco y negro)

```
Black and white monochromatic logo for SIRAN drone detection system. Geometric eye with crosshair and drone silhouette. Clean vector style, pure black (#000000) on white (#FFFFFF). Suitable for print, grayscale documents and official thesis. No gradients, no color fills. Text "SIRAN" in bold sans-serif below the symbol.
```

---

## Prompt 7 — Versión fondo oscuro (dark mode)

```
Dark mode version of SIRAN logo. White geometric eye shape with detection reticle on deep black background #0D0D0D. Simplified quadcopter silhouette inside. Thin white lines, no fills. "SIRAN" text in white Roboto Bold or Montserrat Bold below. Professional, technical, academic aesthetic. No military insignias, no weapons.
```

---

## Prompt 8 — Versión tipo interfaz tecnológica (HUD / terminal)

```
HUD-style tech interface logo for SIRAN drone detection system. Scanline texture, terminal green (#22C55E) color, black background. Geometric eye with detection bounding box overlay and 4-rotor drone shape inside. Corner bracket decorations. Monospace font "SIRAN" with blinking cursor effect. Matrix-inspired but clean and professional. No weapons, no military symbols.
```

---

## Prompt negativo — Elementos a evitar en todos los prompts

```
Negative prompt: no weapons, no guns, no military insignias, no official shields, no national flags, no explosions, no blood, no violence, no army logos, no government seals, no eagles, no anchors, no real military emblems, no institutional crests, no aggressive imagery, no photorealism, no 3D render, no lens flare, no complex gradients, no clip art, no watermarks.
```
